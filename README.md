# web_supertuti_eco
Supermercado Tuti - Ecotec

## Descripción del proyecto
Aplicación web que permite gestionar una lista de compras para el supermercado TUTI.

## Funcionalidades
- Agregar productos
- Marcar productos como comprados
- Eliminar productos
- Contador de productos
- Persistencia con localStorage

## Tecnologías usadas
- HTML5
- CSS3
- JavaScript

## Instrucciones de uso
1. Escribir el nombre del producto
2. Ingresar la cantidad
3. Presionar "Agregar"
4. Clic sobre el producto para marcar como comprado
5. Botón X para eliminar
